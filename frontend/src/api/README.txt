# Carpeta api

Aquí se centralizan todas las funciones para consumir la API del backend (solo lectura, GET). Cada archivo representa un recurso o grupo de endpoints. Ejemplo: `ingredientes.ts` contiene la función para obtener los ingredientes y mapea los datos al formato usado en el frontend.

- http.ts: configuración base de axios.
- ingredientes.ts: función para obtener ingredientes desde la API y mapear los campos.
