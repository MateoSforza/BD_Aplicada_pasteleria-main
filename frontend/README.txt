Estructura base para el frontend (React + TypeScript)

1. src/ : Carpeta principal del código fuente.
2. src/api/ : Aquí irán los archivos para conectar con la API (por ejemplo, funciones para llamar a endpoints de ASP.NET Web API).
3. src/components/ : Componentes reutilizables de React.
4. src/pages/ : Vistas principales de la aplicación.
5. src/App.tsx : Componente raíz.
6. src/main.tsx : Punto de entrada de la app.

Recuerda instalar las dependencias con:
npm install react react-dom axios
npm install -D typescript @types/react @types/react-dom vite @vitejs/plugin-react

Luego puedes iniciar el proyecto con:
npm run dev
